MdcImageList

Selector: mdc-image-list

Exported as: mdcImageList

Properties

Name Description
masonry: boolean Indicates that this Image List should use the Masonry variant.
textProtection: boolean Indicates that supporting content should be positioned in a scrim overlaying each image (instead of positioned separately under each image).

MdcImageListItem

Mandatory. Indicates each item in an Image List.

Selector: mdc-image-list-item mdcImageListItem

Exported as: mdcImageListItem

MdcImageListImageAspect

Optional. Parent of each item's image element, responsible for constraining aspect ratio. This element may be omitted entirely if images are already sized to the correct aspect ratio.

Selector: mdc-image-list-image-aspect mdcImageListImageAspect

Exported as: mdcImageListImageAspect

MdcImageListImage

Mandatory. Indicates the image element in each item.

Selector: mdc-image-list-image mdcImageListImage

Exported as: mdcImageListImage

MdcImageListSupporting

Optional. Indicates the area within each item containing the supporting text label, if the Image List contains text labels.

Selector: mdc-image-list-supporting MdcImageListSupporting

Exported as: mdcImageListSupporting

MdcImageListLabel

Optional. Indicates the text label in each item, if the Image List contains text labels.

Selector: mdc-image-list-label mdcImageListLabel

Exported as: mdcImageListLabel
