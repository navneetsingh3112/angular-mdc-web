Service

MdcDialog

Service to open MDC dialogs.

Properties

Name Description
afterDismiss: Observable 
R | undefined
 Gets an observable that is notified when the dialog is finished closing.

Methods

close(dialogResult?: any) Closes the dialog with optional returned result.
open(component, config?: MdcDialogConfig) Opens the dialog with optional configuration.

MdcDialogConfig

id: string ID for the dialog. If omitted, a unique one will be generated.
ariaLabel: string Aria label to assign to the dialog element.
ariaDescribedBy: string ID of the element that describes the dialog.
clickOutsideToClose: boolean Close the dialog if user clicks on the backdrop. (Default: true)
escapeToClose: boolean Enable or disable close on escape key (Default: true).
scrollable: boolean Applied automatically when the dialog has overflowing content to warrant scrolling.
buttonsStacked: boolean Enable or disable the stacking of action buttons, if they can't fit on a single line.
data: any Data that can be assigned inside child component.

Directives

MdcDialogContainer

Exported as: mdcDialogContainer

MdcDialogSurface

Mandatory. The bounding box for the dialog's content.

Exported as: mdcDialogSurface

MdcDialogTitle

Mandatory. The bounding box for the dialog's content.

Exported as: mdcDialogTitle

MdcDialogContent

Optional. Primary content area. May contain a list, a form, or prose.

Selector: mdc-dialog-content mdcDialogContent

Exported as: mdcDialogContent

MdcDialogActions

Optional. Footer area containing the dialog's action buttons.

Selector: mdc-dialog-actions mdcDialogActions

Exported as: mdcDialogActions

MdcDialogButton

Exported as: mdcDialogButton
